## üé¨ Production Tips

### For Full Demo (3min)
1. **Recording:**
   - Use Loom or OBS Studio
   - 1920x1080 resolution minimum
   - Clean browser window (no bookmarks bar, close extra tabs)
   - Use incognito mode for clean UI

2. **Voiceover:**
   - Use a good USB microphone
   - Script should sound natural, not robotic
   - Re-record sentences that sound off
   - Add subtle background music (low volume, non-distracting)

3. **Editing:**
   - Use DaVinci Resolve (free) or Final Cut Pro
   - Add subtle zoom-ins on important UI elements
   - Use smooth transitions
   - Add text overlays for key stats

4. **Upload:**
   - YouTube (public or unlisted)
   - Title: "Veo-licious Gems - AI Marketing Agency-in-a-Box | GDG Stanford Hackathon"
   - Description: Include all links (demo, GitHub, team info)
   - Tags: Google Gemini, Veo, AI, Hackathon, Startup, Marketing, SMB

### For Viral Teaser (30s)
1. **Recording:**
   - Vertical format (9:16 for TikTok/Reels)
   - Fast cuts (2-3 seconds max per scene)
   - Use on-screen text overlays
   - Add trending sound/music

2. **Hook:**
   - First 3 seconds MUST grab attention
   - Use big, bold text
   - Show the "wow" factor immediately

3. **Posting Strategy:**
   - Post on Twitter, LinkedIn, TikTok, Instagram Reels
   - Ask team + friends to share in first hour
   - Reply to every comment to boost engagement
   - Cross-post in relevant communities (r/entrepreneur, r/smallbusiness, etc.)

4. **Timing:**
   - Post during Phase 1 of judging (during hackathon)
   - Repost daily during Phase 2 (7-day tracking period)
   - Boost with $50-100 ad spend if needed
